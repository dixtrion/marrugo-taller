# Sistema Técnico Marrugo 🛠️

Calculadora profesional de taller para metalmecánica y sistemas de escape.
Desarrollado para la conversión rápida de circunferencias a pulgadas comerciales (fracciones) y detección de calibres.

### Funciones:
- Conversión de CM a Pulgadas decimales y fracciones (ej: 1.75" -> 1 3/4").
- Identificador de Calibres (Ga) por milímetros.
- Diagnóstico de material y recomendación de zapata.
- Herramientas geométricas de precisión.
